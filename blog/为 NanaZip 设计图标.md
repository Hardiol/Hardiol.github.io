## 概述 

NanaZip 是一个开源文件归档器，旨在提供现代 Windows 体验，从著名的开源文件归档器 7-Zip 的源代码派生而来。  这也是世界上第一个支持 Windows 11 新版右键菜单 API 的应用。我与毛利目标相同，都旨在推动 Windows 应用的现代化。
